## Project Overview
CapeTownMagazine is a website designed to showcase Cape Town’s culture, attractions, and lifestyle. The project demonstrates HTML structure, navigation, content organization, advanced responsive styling, and client-side interactive scripting. Images and references are used to make the content authentic and visually engaging.

## Goals and Objectives
- Provide a clear online guide to Cape Town’s attractions and events.  
- Meet the requirements for all parts of the assignment, including advanced programming and validation.  
- Establish a complete client-side application optimized for user interactivity and smooth responsive design.  

## Key Features
- **Home:** Latest articles and featured content.  
- **About Us:** Background, vision, and readership.  
- **News:** City developments, events, and community updates.  
- **Things To Do:** Attractions, outdoor activities, and cultural experiences with images and an interactive lightbox.  
- **Membership / Order Form:** Subscription form featuring custom client-side validation and simulated asynchronous submission.  
- **Contact Us:** Social links, email, phone, and embedded map.

## Deliverables (Part 2)

## Deliverables (Part 3)
- Fully functional `script.js` file organizing form, search, and gallery logic.
- Client-side form validation tracking missing data or invalid emails using custom regex.
- Custom inline error feedback injected dynamically under failing input nodes without native browser alert popups.
- Asynchronous form handling simulation using `preventDefault()` and `setTimeout` to route successful submissions to `thanks.html`.
- Global live-matching search input panel matching keywords to append interactive drop-down navigation links.
- Pure programmatically created lightbox overlay canvas generated strictly within the DOM script layer upon image click event.
- Anchor tag tracking system displaying live destination paths across the baseline status bar on mouse hover.
